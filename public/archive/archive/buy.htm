To order from Falcom, you first need to send an e-mail to nihon@falcom.com similar to this:


Dear Nihon Falcom:

I would like to order:

ITEMNUMBER (this is different from the catalog number, for the FMS series, it is in the format MXXX, for example M029--check Falcom's CD Page for the number)
ITEMNAME
ITEMPRICE

(Repeat the above for each item - You can get the above info from Falcom's CD Page)

TOTALPRICE

Name: NAME
Address: ADDRESS

Credit Card Type: CCTYPE
Credit Card #: CCNUMBER
Expiration Date: CCDATE

Thank you very much.

Sincerely,
NAME
DATE


Then, you need to wait for a response to it asking if the order is ok. It might be phrased awkwardly, like this:


Thank you for an inquiry of Falcom mail order service.
The total price is as follows.

ITEMPRICE: ITEMNUMBER ITEMNAME

(Repeat for every item)

SHIPPINGCOST
TOTALPRICE

If you are ok, we think we would like to accept your order.
We will be looking forward to contacting from you again.


Guess what? You send them another e-mail like this:


The order is OK.

Thank you very much.

Sincerely,
NAME
DATE


Shortly, you should get an e-mail like this:


Thank you for your reply.

We accepted your order.
The total price is as follows.

ITEMPRICE: ITEMNUMBER ITEMNAME

(Repeat for every item)

SHIPPINGCOST
TOTALPRICE

It is your receipt number [RECEIPTNUMBER].
Please write the number of receipts, and send an E-mail if there is a question.
We will send a product on SHIPDATE.
After a product is sent, the slip number of EMS will be reported.
Be waiting, please.

Thank you.

Nihon Falcom Mailorder Service


When they ship the package, you should get an e-mail like this:


Thank you for using falcom mail order service.
We sent product in SHIPDATE.

By EMS (International Express Mail)
EMS item number : EMSITEMNUMBER

The total price is as follows.

ITEMPRICE: ITEMNUMBER ITEMNAME

(Repeat for every item)

SHIPPINGCOST
TOTALPRICE

We'll be looking forward to ordering from you again.

Thank you


Remember that the person who is reading this doesn't speak English well (obviously), so don't use complex grammar or vocabulary. K.I.S.S.. If you have any questions about this process, feel free to e-mail me at quasireality@yahoo.com.
