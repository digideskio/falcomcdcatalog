Falcom CD Catalog - English Edition

Introduction

The purpose of this website is to provide English tracklist translations for all Falcom albums and redbook game soundtracks. This website aims to be an English version of Falcom's CD Catalog, although there are some differences as noted below. It is not an official Falcom website, but it is one of Falcom's Main Homepages Abroad.

There are some differences between Falcom's CD Catalog and this one. The albums are organized by series and each is given its own page, unlike the frameset with multiple albums per page used by Falcom. In addition, some spelling errors and track omitions in Falcom's catalog are corrected here. This catalog also features some promotional albums that are not in Falcom's catalog.

The Falcom Millenium Series albums are the only albums that are still being produced, and they can be purchased at Cocoebiz or directly from Nihon Falcom. Older albums may still be available at CD Japan, Game Music Online, or another importer, but many of them are impossible to find except on Yahoo! Japan Auctions, EBay, or another auction site.
